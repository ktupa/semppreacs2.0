# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.2.0] - 2026-01-16

### Adicionado
- Sistema de atualização automática para clientes
- API para gerenciamento de updates (`/api/updates`)
- Script de instalação para novos clientes
- Sistema de backup automático antes de atualizações
- Proteção de arquivos de configuração durante updates
- Rollback automático em caso de falha

### Melhorado
- Documentação de instalação e configuração
- Sistema de versionamento

## [1.1.0] - 2026-01-10

### Adicionado
- Mobile API para integração com apps
- Análise de conectividade com ML
- Dashboard de métricas avançadas

## [1.0.0] - 2026-01-01

### Adicionado
- Versão inicial do SemPPRE Bridge
- Integração com GenieACS
- Integração com IXC Provedor
- Dashboard de dispositivos TR-069
- Sistema de autenticação
- Provisioning automático
